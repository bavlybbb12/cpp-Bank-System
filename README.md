\# 🏦 C++ Banking System as an application for an oop course



A secure, console-based banking application that simulates real-world financial operations.



\## 🚀 Features

\* \*\*Admin Panel:\*\* Create accounts, delete accounts, and manage interest rates.

\* \*\*User Portal:\*\* Deposit, withdraw, and \*\*transfer funds\*\* between users.

\* \*\*Security:\*\* Prevents duplicate IDs and overdrafts.

\* \*\*Persistence:\*\* Saves all data to file automatically.
